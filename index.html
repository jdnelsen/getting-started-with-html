Lesson 0 Basics of the Web and HTML

What is a Web Page- A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

How Coding Works - Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote. Programmers need to write exactly the way a computer understands

HTML stands for: Hypertext Markup Language. HTML is the main type of document on the web.

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag, Some tags (called "void" tags) do not require a closing tag

Element: An HTML element refers to everything within a set of opening and closing tags. Some elements are for inline content while others for block. Inline simply means that the content is viewed as being on the same line or lines, while block content has padding around it placing it in its own invisible content box. Every element has a box.

Span are inline and DIV are block

Attribute: This is a property of an HTML element.
